# Bookstore

This is a Book Store project which displays Books by their ID, name, buyer name, and date. They can be sorted, deleted, edited, added, and fetched. 

* **Front End** : React.JS, MUI (Datagrid)
* **Back End** : Spring Boot (REST API)
* **Database** : MongoDB
